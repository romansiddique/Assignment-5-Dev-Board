what the difference among innerHTML, innerText and textContent

The difference among innerHTML, innerText and textContent is:

1. innerText return only tag content.
2. innerHTML return content including tags.
3. textContent return content including hidden text.

what is event delegation in DOM

The event delegation in DOM is:

Event delegation is a technique that helps you to control event bubbling.

How to use create, add, remove element in JS

The uses of create, add and remove element is:

1. create use to create an element.
example:
const createParagraphTag = document.body.createElement('p')
It will create a paragraph tag.

2. add use to add class in any tag.
example:
document.getElementById("id").classList.add('class')

3. remove use to remove an element directly.
example:
createParagraphTag.remove()
It will remove this paragraph tag.
